# CloudFlare Quick Setup Guide

## Initial Setup Steps

### 1. Account Creation
```yaml
URL: https://dash.cloudflare.com/sign-up
Requirements:
  - Email address
  - Password
  - No payment needed
```

### 2. Domain Addition
```yaml
Steps:
1. Click "Add site"
2. Enter: ihelper.tech
3. Select: Free Plan
4. Continue through setup

DNS Settings:
  Type: A
  Name: @
  Content: [GitHub Pages IP]
  Proxy: Yes
```

### 3. Pages Configuration
```yaml
Setup:
1. Go to Pages
2. Connect to GitHub
3. Select: ihelper-tech
4. Configure build:
   - Framework: None
   - Build command: None
   - Output: /
```

### 4. Quick Verification
```yaml
Check:
1. DNS propagation
2. SSL/TLS active
3. Pages deployed
4. Site accessible
```
